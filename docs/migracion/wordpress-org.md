# Plan De Migracion A WordPress.org

## Cuando migrar

Migrar a WordPress.org y hosting propio cuando existan senales claras de validacion:

- Volumen consistente de registros.
- Necesidad real de funcionalidades no disponibles en WordPress.com.
- Interes de empresas o aliados.
- Necesidad de mayor control sobre datos, plugins, integraciones o rendimiento.

## Beneficios de migrar

- Control total del hosting.
- Instalacion libre de plugins.
- Tema propio o personalizado.
- Integraciones avanzadas.
- Mejor control de datos y automatizaciones.
- Posibilidad de construir funcionalidades a medida.

## Pasos sugeridos

1. Elegir hosting.
2. Instalar WordPress.org.
3. Exportar contenido desde WordPress.com.
4. Importar paginas, entradas y medios.
5. Configurar tema base.
6. Instalar plugins esenciales.
7. Migrar formularios y datos.
8. Configurar dominio y SSL.
9. Probar sitio completo.
10. Redirigir trafico.

## Riesgos a cuidar

- Perdida de registros de formularios.
- Formularios mal configurados.
- Cambios de URLs.
- SEO inicial.
- Privacidad de datos personales.
- Dependencia de plugins innecesarios.

# Arquitectura De Informacion

## Mapa del sitio

- Inicio
- Talentos
  - Registro de Talento
- Empresas
  - Interes Empresas y Aliados
- Sobre TalentoRD
- Contacto
- Politica de Privacidad

## Flujo principal para talentos

1. Llega a Inicio o Talentos.
2. Entiende la propuesta.
3. Hace clic en Registrar mi perfil.
4. Completa formulario.
5. Recibe confirmacion.

## Flujo principal para empresas

1. Llega a Inicio o Empresas.
2. Entiende que TalentoRD esta validando una red de talento.
3. Hace clic en Quiero conocer talento.
4. Completa formulario.
5. Recibe confirmacion o seguimiento.

## Prioridad de conversion

1. Registro de talento.
2. Solicitud de empresas/aliados.
3. Contacto general.

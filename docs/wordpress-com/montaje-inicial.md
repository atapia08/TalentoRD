# Montaje Inicial En WordPress.com

Esta guia describe como construir manualmente la primera version del sitio en WordPress.com.

## 1. Crear sitio

1. Entrar a WordPress.com.
2. Crear un nuevo sitio con el nombre TalentoRD.
3. Elegir un dominio temporal o dominio propio si ya esta disponible.
4. Seleccionar un tema limpio, moderno y facil de editar.

## 2. Configurar paginas

Crear estas paginas:

- Inicio
- Talentos
- Empresas
- Sobre TalentoRD
- Contacto
- Registro de Talento
- Interes Empresas y Aliados

## 3. Configurar menu

Menu principal sugerido:

- Inicio
- Talentos
- Empresas
- Sobre TalentoRD
- Contacto

Boton destacado si el tema lo permite:

- Registrarme

## 4. Formularios

Usar el bloque de formulario disponible en WordPress.com o un plugin/formulario permitido por el plan contratado.

Formulario de registro de talento:

- Nombre completo
- Correo
- Telefono o WhatsApp
- Ciudad / provincia
- Area de talento
- Nivel de experiencia
- Enlace a portafolio, LinkedIn o CV
- Que tipo de oportunidad busca
- Comentario adicional
- Aceptacion de contacto y tratamiento de datos

Formulario para empresas y aliados:

- Nombre de contacto
- Empresa / institucion
- Correo
- Telefono
- Tipo de interes
- Areas de talento que busca
- Mensaje

## 5. Ajustes basicos

- Configurar titulo del sitio: TalentoRD.
- Configurar descripcion corta: Talento dominicano conectado con oportunidades.
- Revisar idioma, zona horaria y enlaces permanentes.
- Crear politica de privacidad basica.
- Activar comentarios solo si aportan al objetivo.
- Conectar analitica disponible segun el plan.

## 6. Publicacion

Antes de publicar:

- Revisar textos en movil y escritorio.
- Probar formularios.
- Confirmar que los correos llegan.
- Revisar enlaces del menu.
- Publicar y compartir con un grupo pequeno para validacion.

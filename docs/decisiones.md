# Registro De Decisiones

## 2026-06-06

Decision:

Usar WordPress.com para montar manualmente una primera version simple del sitio y validar la idea antes de migrar a WordPress.org.

Motivo:

Reduce costo, complejidad y tiempo de lanzamiento. Permite aprender con paginas y formularios antes de desarrollar funcionalidades avanzadas.

Implicacion:

El repositorio se usara para documentacion, diseno, funcionalidades y preparacion tecnica, no como fuente directa del sitio en esta fase.
